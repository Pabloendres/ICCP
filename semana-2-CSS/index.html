Selectores

Existen principalmente tres tipos de selectores:

  1. Selectores directos
  2. Selectores de id
  3. Selectores de clase

Selectores directos

Los selectores directos se utilizan para modificar a todos los elementos del HTML, siguiendo la jerarquía del documento. Estos se utilizan seleccionando el elemento que se quiere modificar. Por ejemplo, para modificar a todos los elementos <p> con texto de color rojo, podemos escribir: 

p {
    color: red;
}

Otro ejemplo son algunos de los elementos de este mismo documento. ¡Revisa el código!

Selectores de id

Estos selectores afectan a un único elemento en el documento, aquel son el que concuerde el atributo id.

Por ejemplo, este párrafo en particular tiene id="pEstiloso" y puede ser modificado con el siguiente selector, anteponiendo # al id del elemento: 

#pEstiloso {
    color: blue;
}

Selector de clase

Un punto medio entre los selectores directos, que modifican todos los elementos, y los de id, que modifican a sólo uno, son los selectores de clase. Estos selectores modificarán a todos los elementos que tengan una clase determinada.

Por ejemplo, estos últimos dos párrafos pertenecen a la clase clase="clasep", y se pueden modificar por el sigiuente selector, , anteponiendo . a la clase del elemento: 

.clasep {
    color: red;
}
